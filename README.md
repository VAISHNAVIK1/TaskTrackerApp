# Task Tracker App (Java + JDBC + MySQL)

A simple console-based Task Tracker application built using Core Java, JDBC, and MySQL. It allows users to add tasks, view them, mark tasks as completed, and delete tasks. The project follows DAO structure and uses PreparedStatements to prevent SQL injection.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Uses PreparedStatement (SQL injection safe)
- Simple and clean code structure

## Tech Stack
- Java (JDK 21)
- JDBC
- MySQL
- Eclipse IDE
- Git & GitHub
